# Changelog

## 0.8.0 - 2026-03-16

Initial launch of the `miounet11/llmfit` fork as a user-facing open-source
project.

### Added

- Product-first README and positioning for local AI builders and platform teams.
- Static marketing and docs site in `site/`.
- Docker Compose stack for running the API and site together.
- Isolated deployment assets for a separate site container.
- Community files: contributing guide, security policy, issue templates, and PR template.

### Changed

- Retargeted repository metadata, installer URLs, and package metadata to `miounet11/llmfit`.
- Simplified release automation to focus on GitHub release assets and GHCR images.
- Removed upstream-specific packaging artifacts that were not relevant to the public fork.
